## Detailed Description (max 16,000 characters)

**Transform any webpage into an editable canvas with Web Screen Draft.**

Web Screen Draft is the ultimate tool for legal professionals, product teams, compliance officers, and anyone who needs to modify web content visually and capture the results. Whether you're documenting evidence, creating mockups, or preparing compliance reports, Web Screen Draft makes it effortless.

**🎯 Key Features:**

**Visual Page Editing**
• Click any element to select and drag it anywhere on the page
• Switch to text mode to select specific words or phrases
• Resize images with 8-handle controls (corners for proportional, edges for free scaling)
• Double-click text overlays to edit content inline

**Clipboard Integration**
• Paste text or images directly from your clipboard onto any webpage
• Seamlessly integrate external content into your page modifications

**Professional Screenshots**
• Capture viewport or full-page screenshots
• Side-by-side comparison screenshots (before/after)
• Clean captures that hide editing UI elements automatically
• Optional red highlighting for important elements

**Workflow Features**
• Keyboard shortcut (Ctrl/Cmd+Shift+E) for quick toggle
• One-click reset to restore original page state
• All changes persist until page reload or manual reset

**🏢 Perfect For:**
• **Legal Teams**: Document evidence, capture page states for litigation
• **Compliance Officers**: Create visual compliance reports and documentation
• **Product Teams**: Create mockups and document feature requests
• **UX Designers**: Test layout changes and capture variations
• **Content Managers**: Visualize content changes before implementation

**🔒 Privacy First:**
• All processing happens locally in your browser
• No data transmitted to external servers
• No account required - works offline

**✨ Easy to Use:**
1. Click the Web Screen Draft icon in your toolbar
2. Enable edit mode on any webpage
3. Select, move, paste, and modify content visually
4. Capture screenshots of your modifications
5. Reset when done - it's that simple!

Web Screen Draft respects your privacy and works entirely within your browser. Perfect for sensitive legal work, confidential compliance documentation, and professional use cases where data security matters.
