# Privacy Policy for Web Screen Draft

**Last updated: March 19, 2026**

## Information We Collect

Web Screen Draft is a browser extension that operates entirely locally on your device. We do not collect, store, or transmit any personal information.

## Data Usage

### Clipboard Access
- **What**: The extension reads your clipboard content when you use the paste feature (Ctrl/Cmd+V)
- **Why**: To paste text or images directly onto web pages as part of the editing functionality
- **Storage**: Clipboard data is only accessed temporarily and is not stored or transmitted anywhere

### Screenshot Capture
- **What**: The extension captures screenshots of your browser tab when you use screenshot features
- **Why**: To provide the core screenshot functionality
- **Storage**: Screenshots are saved locally to your Downloads folder and are not transmitted anywhere

### Website Content Access
- **What**: The extension can read and modify the content of web pages you visit
- **Why**: To provide editing functionality (moving elements, adding overlays, etc.)
- **Storage**: All modifications are temporary and stored only in browser memory until you reload the page

## Data Storage

All data processed by Web Screen Draft remains on your local device:
- No data is transmitted to external servers
- No analytics or tracking are performed
- No user accounts or cloud storage are used

## Third-Party Libraries

Web Screen Draft includes the html2canvas library for screenshot functionality. This library operates entirely within your browser and does not transmit data externally.

## Changes to This Policy

We may update this privacy policy occasionally. Any changes will be reflected in the "Last updated" date above.
